Definition of a group

A group is a type of algebraic structure. It is made up of a set S (a collection of objects), a binary operation ∘, and must follow these three axioms

  1. Elements of the set must be associative under the binary operation
  2. There exists an element i in the set S such that for all elements s in the set S this statement holds: s ∘ i = s = i ∘ s. We call the element i the identity element
  3. For all elements s in the set S there exists an element s-1 belonging to S such that the following statement holds: s ∘ s-1 = i = s-1 ∘ s. We call the element s-1 the inverse element of s

The definition I have just given might sound confusing however a quick example should show you that groups are actually quite simple.

A quick example of a group

For this group we will use the set {..., -2, -1, 0, 1, 2, ...} i.e. the integers and the binary operation of addition. Does this satisfies the three axioms?

  1. As you all know for any three real numbers x, y, z addition is associative: (x + y) + z = x + (y + z)
  2. The integer that satisfies the second axiom is 0. Take any integer x then x + 0 = x = 0 + x. Hence 0 is the identity element
  3. Take an integer a then its inverse element is -a

Hence the integers and addition for a group.

Another example of a group

For this example we will be studying a finite group. The set is integers modulo 3 with the binary operation being addition. As our example only exists of three elements we describe the structure of the group using a Cayley table.
